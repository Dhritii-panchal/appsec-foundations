# Linux Security Basics

## Commands Used
- ls, cd, pwd
- chmod, chown
- ps, top
- netstat, ss

## File Permissions
- Read, Write, Execute
- User, Group, Others

## Security Perspective
- Linux permissions control access to files
- Misconfigured permissions can lead to privilege escalation
- Understanding processes and services is critical for security analysis

## Exposure Awareness
- SUID binaries
- Running services
- Open ports
